第一章：初识Python

print 输出函数

print函数是Python中最基础的输出函数，用于在控制台显示信息。

语法结构：

基本语法
print(输出内容)

print() 函数完整语法格式：

完整语法
print(*values, sep=" ", end="\n", file=None, flush=False)

示例1：基本输出

示例1：输出字符串
print("hello world")
hello world
示例2：输出数字
print(123)
123

示例2：使用分隔符

使用默认分隔符
print(1, 2, 3, 4) # 1 2 3 4 默认分隔符为空格
1 2 3 4
自定义分隔符
print(1, 2, 3, 4, sep="->") # 分隔符改为了箭头
1->2->3->4

示例3：控制结束符

默认换行
print(1, 2, 3) # print 默认为\n 代表换行 print(1, 2, 3)
1 2 3 1 2 3
使用制表符
print(1, 2, 3, end="\t") # \t为制表符、为一个tab的大小 print(1, 2, 3)
1 2 3 1 2 3

示例4：输出到文件

使用文件对象
# 打开名为"报错.txt"的文件，以追加模式("a")打开 w为写入 会覆盖原文本 # 如果文件不存在，会自动创建 f = open("报错.txt", "a") print("hello world", file=f) # 将字符串"hello world"写入文件中 f.close() # 关闭文件，确保所有写入的内容保存到文件中
使用with语句（推荐）
# 使用 with 语句打开文件"1.txt"，以追加模式("a")打开 # with 语句可以确保文件在操作完成后自动关闭，避免手动关闭 with open('1.txt', 'a') as f: print("print语句需要加上括号111", file=f) # 文件会在with语句结束时自动关闭
提示： 使用 with 语句操作文件是一种更安全的做法，它会自动处理文件的关闭，即使发生异常也能确保文件被正确关闭。

格式化输出

含义

给定一个模板，按照模板输出内容。

格式符号

格式化输出使用 % 和不同的字符连用，不同类型的数据需要使用不同的格式化字符。

常用格式化字符及其含义

格式化字符 含义
%c 字符
%s 字符串
%d 十进制整数
%f 浮点数

三种格式化方法

1. 基本格式化输出：使用%

%格式化
print('我的名字是%s，年龄是%d岁' % ("01", 18))
我的名字是01，年龄是18岁

2. format()格式化输出

format()方法
print("我的名字是：{}，年龄为：{}岁".format("01", 18))
我的名字是：01，年龄为：18岁

3. f表达式格式化输出（Python 3.6+）

f-string(推荐)
name, age = "01", 20 # name = "01" # age = 18 print(f"名字是{name},年龄是{age}岁")
名字是01,年龄是20岁
提示： f-string（格式化字符串字面量）是Python 3.6引入的新特性，它提供了一种简洁、方便的格式化方式，是目前推荐的字符串格式化方法。

变量

变量概念

  • 变量是将数据在内存中存储后给定的一个名称，这个名称就是变量。
  • 变量名是没有类型的，类型只存在于对象中，变量只是引用对象。
  • Python 为动态解释型语言，在赋值操作时，类型是在运行过程中自动决定的。

对象

  • 对象是有类型的。
  • 对象是分配的一块内存空间，用来表示它的值。
  • Python 对象的三要素：
    • id：对象的唯一标识符。
    • type：对象的类型。
    • value：对象的值。
定义变量
a = 1 # 定义变量a print(a)
1
变量的id（内存地址）
a = 123 print(id(a)) a = 12 print(id(a))
2095699744944 2095699552912
注意： 从输出可以看出，当变量a的值改变时，其id（内存地址）也发生了变化，说明Python创建了一个新的对象。
实际应用
epoch = 100 acc = 0.8 print(f"训练的总轮数为{epoch}，得到的acc为{acc}")
训练的总轮数为100，得到的acc为0.8

变量的命名规则

  • 变量名只能由数字、字母和 _(下划线) 组成。
  • 变量名不能以数字开头。
  • 变量名不能是 Python 关键字。
  • 变量名是区分大小写的。
Python命名规范：
  • 变量和函数名使用小写字母，单词之间用下划线分隔（snake_case）
  • 类名使用大驼峰命名法（CapWords）
  • 常量通常使用大写字母，单词之间用下划线分隔

输入函数

input() 函数

input() 是 Python 中用于从用户获取输入的函数。它会在控制台显示提示信息，并等待用户输入。用户输入的内容会以字符串形式返回。

基本用法

基本输入
a = input("请输入一些内容：") print("你输入的内容是:", a) print(type(a)) # 字符串类型
请输入一些内容：123 你输入的内容是: 123
注意： input() 函数始终返回字符串类型，即使用户输入的是数字。

类型转换

如果需要将用户输入解释为其他类型，需要进行类型转换：

转换为整数
age = int(input("请输入你的年龄：")) print("你输入的内容是:", age) print(type(age)) # 整型
请输入你的年龄：123 你输入的内容是: 123
常用类型转换函数：
  • int(): 转换为整数
  • float(): 转换为浮点数
  • str(): 转换为字符串
  • bool(): 转换为布尔值
注意： 当用户输入的内容无法转换为目标类型时，会引发 ValueError 异常。例如，如果用户在提示输入年龄时输入了"abc"，int() 将无法将其转换为整数，程序会崩溃。实际开发中，应该使用异常处理来处理这种情况。
