后端做了什么

  • 只接受白名单 id / group / list，不允许前端传任意外部 URL；all 仅对受控服务端监控开放。
  • 统一设置请求头、超时、重试、缓存和 JSON/XML 文本解析。
  • 按访客限制 Function 请求与上游调用预算，批量路由另有十分钟频率限制。
  • 未来需要 Key 的 API 只读 Netlify 环境变量，不写进前端代码。
