Git基础命令

仓库初始化

# 初始化新仓库
git init

# 克隆现有仓库
git clone https://github.com/user/repo.git

提交更改

# 添加所有修改到暂存区
git add .

# 提交到本地仓库
git commit -m "提交说明"

分支管理

# 创建新分支
git branch feature-branch

# 切换分支
git checkout main

# 合并分支
git merge feature-branch

常用工作流

  • 集中式工作流
  • 功能分支工作流
  • Git Flow工作流

高级技巧

# 修改最近提交
git commit --amend

# 交互式变基
git rebase -i HEAD~3
